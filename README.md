# ⚔️ FFXIV 副本开荒犯错记录表

一个面向 FFXIV 静态队的在线犯错记录与复盘工具。队长记录犯错，队员添加复盘备注，一键生成群公告总结。

**[📍 在线演示](https://shiraishiyokai.github.io/ffxiv-raid-tracker/ffxiv-raid-tracker.html)**

---

## ✨ 功能特性

| 功能 | 说明 |
|------|------|
| ⚡ 快速记录 | 战斗间隙 3 秒录入一条犯错，支持键盘快捷键 `1`-`8` 选队员 |
| 🎯 预设模板 | 一键添加常见犯错（踩圈/吃AOE/换T晚等），可自定义 |
| 💬 复盘备注 | 队员点击犯错条目展开，添加复盘 tips |
| 📋 一键总结 | 生成格式化当日总结，复制直接贴群公告 |
| 🔗 在线共享 | 基于 jsonblob.com 免费云存储，无需服务器，即开即用 |
| 👑 权限分离 | 队长链接可编辑，队员链接仅可添加备注，通过 URL 区分 |
| 🗓 迷你日历 | 有记录的日期绿色标记，凌晨 4 点前算前一天 |
| ⚠️ 团队问题 | 记录减伤不足/奶不上等无法归属个人的问题 |
| 🛡️ 职业图标 | 8 名队员对应 FFXIV 职业图标（绝枪/骑士/白魔/学者/武士/钐镰/机工/赤魔） |
| 💾 数据安全 | 本地 localStorage + 云端双重保存，支持 JSON 导出/导入 |

---

## 🚀 快速开始

### 队长

1. 打开页面，网址末尾加 `#captain` 进入队长模式
2. 点击 **🆕 创建在线记录本** → 获得记录本 ID
3. 点击 **📋 复制链接** → 把队长链接和队员链接发到群里
4. 开荒时用快速记录栏录入犯错，结束后生成总结

### 队员

1. 打开队员链接（不带 `#captain`）
2. 点击犯错条目的 ▶ → 填写复盘备注 → 自动保存

|------|------|------|

| 🧑 队员链接 | `...html` | 查看记录、添加复盘备注 |

## ⌨️ 快捷键

| 按键 | 功能 |
|------|------|
| `1`-`8` | 选择 MT/ST/H1/H2/D1/D2/D3/D4 |
| `T` | 选择团队问题 |
| `Enter` | 提交快速记录 |
| `Esc` | 关闭弹窗 |

## 👥 多队伍支持

每个「在线记录本」就是一支独立队伍，互不干扰。创建新记录本即可供另一支队伍使用。

---

## 🗓 日期说明

- **凌晨 4:00 前仍记为前一天**，确保深夜打本的记录归入同一天
- 日历中有记录的日期标绿色圆点，点击可快速跳转

## 🛠 技术架构

- **单文件 HTML**：CSS + JS 全部内联，零依赖，浏览器直接打开
- **职业图标**：Base64 内嵌，不依赖外部资源
- **云存储**：[jsonblob.com](https://jsonblob.com) 免费 REST API，无需账号
- **本地缓存**：localStorage 持久化 + 离线回退
- **自动同步**：15 秒轮询云端数据
