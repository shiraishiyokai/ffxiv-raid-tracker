# ⚔️ FFXIV 副本开荒犯错记录表

一个面向 FFXIV 静态队的在线犯错记录与复盘工具。队长记录犯错，队员添加复盘备注，一键生成群公告总结。

**[📍 在线使用](https://shiraishiyokai.github.io/ffxiv-raid-tracker/ffxiv-raid-tracker.html)**

---

## ✨ 功能特性

| 功能 | 说明 |
|------|------|
| ⚡ 快速记录 | 战斗间隙 3 秒录入一条犯错，支持键盘快捷键 `1`-`8` 选队员 |

## 🚀 使用方法

### 队长（首次使用）

1. 打开页面，网址末尾加 `#captain` 进入队长模式：
   ```
   https://shiraishiyokai.github.io/ffxiv-raid-tracker/ffxiv-raid-tracker.html#captain
   ```
2. 页面自动创建新队伍，URL 会变为 `?team=xxx#captain`
3. **收藏这个 URL**，以后直接打开就是你的队伍
4. 点击 **📋 复制分享链接** → 把队员链接发到群里

### 队长（日常使用）

1. 打开之前收藏的队长链接
2. 选择日期（默认今天，凌晨 4 点前自动归为前一天）
3. 在快速记录栏录入犯错：
   - **方式一**：点击队员按钮 `[MT]` → 输入描述 → `Enter`
   - **方式二**：点击预设按钮 `[+踩圈]` → 自动添加到选中队员
   - **方式三**：键盘 `1`-`8` 选队员 → 输入描述 → `Enter`
4. 点击 **☁️ 保存到云端** 确保数据同步
5. 开荒结束后点击 **📋 生成当日总结** → 复制到群公告

### 队员

1. 打开队长发的队员链接（不带 `#captain`）
2. 点击犯错条目的 ▶ → 填写复盘备注
3. 点击 **☁️ 保存到云端** 确保备注同步

### 新建队伍

- 队长点击连接栏的 **🆕 新建队伍** 按钮
- 旧队伍数据保留在云端，可通过旧链接随时切换回来

## 🔗 链接权限

| 链接 | 格式 | 权限 |
|------|------|------|
| 👑 队长链接 | `...html?team=xxx#captain` | 记录/删除犯错、修改名字/职业、生成总结、新建队伍 |
| 🧑 队员链接 | `...html?team=xxx` | 查看记录、添加复盘备注 |

## 💾 数据同步说明

| 操作 | 说明 |
|------|------|
| 队长保存 | 点击「☁️ 保存到云端」→ 本地 + 云端双重保存 |
| 队员保存 | 点击「☁️ 保存到云端」→ 自动合并备注到云端最新数据 |
| 自动刷新 | 队长每 15 秒、队员每 2 分钟自动拉取云端最新数据 |
| 离线可用 | 网络断开时使用本地数据，联网后手动保存即可 |

> 💡 队员保存时会先拉取云端最新数据，再叠加备注改动，不会覆盖队长的记录。

## 🛠 技术架构

- **单文件 HTML**：CSS + JS 全部内联，零依赖，浏览器直接打开
- **职业图标**：21 个 Base64 内嵌，不依赖外部资源

## ❓ 常见问题

### Q: 队员看不到最新数据？
确保队长已点击「☁️ 保存到云端」。队员页面会自动刷新，也可手动刷新浏览器。

### Q: 如何切换队伍？
队长点击「🆕 新建队伍」创建新队伍。旧队伍通过收藏的链接随时切换。

### Q: 数据会丢失吗？
数据同时保存在本地 localStorage 和云端 GitHub。刷新页面会自动拉取云端最新数据。

### Q: 不想用默认职业图标怎么办？
队长在成员卡片中点击职业下拉菜单即可切换职业，图标自动更新。
